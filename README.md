# questionnaire-mock

项目链接：
- mock：GitHub - zo-no/questionnaire-mock: 访问卷星项目的mock程序
- 主程序（B端）：GitHub - zo-no/questionnaire-system: 模仿问卷星的项目
- c端：GitHub - zo-no/questionnaire-client: 仿问卷星的客户C端项目，使用nextjs

仿问卷星项目的mock程序

```c
//src是工作目录
mock

```

## 项目启动

npm install
